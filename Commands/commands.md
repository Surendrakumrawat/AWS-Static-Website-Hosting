---

## AWS S3 Console Steps

1. Open the AWS Management Console.
2. Go to Amazon S3.
3. Create a new S3 bucket.
4. Upload the website files (index.html).
5. Enable Static Website Hosting.
6. Disable Block Public Access.
7. Add a Bucket Policy to allow public read access.
8. Copy the Website Endpoint.
9. Open the Website Endpoint in the browser.
10. Verify that the website is live.
